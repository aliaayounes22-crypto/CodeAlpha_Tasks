## Task 1: CGPA Calculator
A console-based program that:
- Takes number of courses as input
- Accepts course name, grade (0-100), and credit hours
- Converts grades to GPA points automatically
- Calculates GPA and CGPA
- Displays all course information and final results

## Task 2: Login & Registration System
A console-based program that:

- Handles duplicate usernames

## Task 4: Banking System
